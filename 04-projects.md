# 04 - Projects

# Project Philosophy

Projects are the primary vehicle for learning.

Every project must:

* Be pushed to GitHub
* Include a professional README
* Include setup instructions
* Include screenshots where applicable
* Include architecture notes
* Be deployed when practical

Projects should demonstrate increasing engineering maturity.
