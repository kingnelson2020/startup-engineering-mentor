# 06 - Career Protocol

# Purpose

Track professional growth alongside technical growth.

---

# CV Protocol

## Initial CV

Create an honest baseline CV.

Document:

* Education
* Current skills
* Existing projects

Do not exaggerate experience.

---

## CV Updates

Update after:

* Major project completions
* Deployments
* New technical competencies

Each update should include:

* Project
* Technologies used
* Impact achieved

---

# LinkedIn Protocol

Keep LinkedIn aligned with:

* CV
* GitHub
* Portfolio

Update after major milestones.

Share:

* Build progress
* Lessons learned
* Project launches

---

# GitHub Protocol

Maintain:

* Consistent commits
* Clear READMEs
* Meaningful commit messages

Pin strongest projects.

Create a professional profile README.

---

# Portfolio Protocol

Portfolio should demonstrate:

* Technical growth
* Product thinking
* Real-world execution

Avoid tutorial clones.

---

# Networking Protocol

Build relationships with:

* Engineers
* Founders
* Product builders
* Startup operators

Focus on genuine engagement.

---

# Job Search Protocol

Track:

* Applications
* Responses
* Interviews
* Feedback

Review patterns monthly.

---

# Continuous Learning Protocol

Technology evolves.

Maintain habits of:

* Reading documentation
* Following industry changes
* Learning new tools
* Reviewing fundamentals

---

# Professional Communication

Practice:

* Technical writing
* Architecture discussions
* Project presentations

Strong communication increases career opportunities.

---

# Long-Term Goal

Progression:

1. Junior Engineer
2. Mid-Level Engineer
3. Senior Engineer
4. Technical Lead
5. Startup Founder or Engineering Leader

The curriculum prepares for the first step while laying foundations for the later stages.
